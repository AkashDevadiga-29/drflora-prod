<h1>Dr.Flora - A Chatbot for plant diseases, agriculture, and gardening.</h1>

This project uses React.js and Bootstrap for the Frontend, Django for the backend, and MongoDB for its database. It uses Tensorflow(Python) and Keras for the model creation with ResNet152V2. 
It also uses OpenAI api to process user queries and LangChain for context filtering. It uses JWT-based authentication using(djangorestframework_simplejwt).

You will need to use the following Django template for the project- 
django-admin startproject example --template https://github.com/mongodb-labs/django-mongodb-project/archive/refs/heads/6.0.x.zip

For further queries for the above setup, go to: https://github.com/mongodb/django-mongodb-backend
